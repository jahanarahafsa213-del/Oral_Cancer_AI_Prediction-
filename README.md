# Oral_Cancer_AI_Prediction-
This project focuses on detecting Oral Cancer using Deep Learning techniques in Google Colab. The model is trained using medical image data to classify oral cancer conditions accurately.

## Project Overview

The main objective of this project is to build an AI-based oral cancer prediction system using image classification techniques. The project includes:

- Data preprocessing
- Image augmentation
- Deep learning model training
- Performance evaluation
- Prediction visualization

---

## Technologies Used

- Python
- Google Colab
- TensorFlow
- Keras
- NumPy
- Pandas
- Matplotlib
- OpenCV

---

## Dataset

The dataset contains oral cancer medical images used for training and testing the model.

Dataset features include:
- Cancerous images
- Non-cancerous images
- Image preprocessing and labeling

---

## Model Architecture

This project uses a Deep Learning CNN model for image classification.

Main steps:
1. Image preprocessing
2. Data splitting
3. Model training
4. Validation
5. Prediction

---

## Results

- Achieved high classification accuracy
- Improved prediction performance using preprocessing techniques
- Visualization of training and validation accuracy

---

## How to Run

1. Open the notebook in Google Colab
2. Upload the dataset
3. Run all cells step by step
4. Train the model
5. Test predictions
